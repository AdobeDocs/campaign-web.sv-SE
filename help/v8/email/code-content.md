# Koda eget e-postinnehåll {#code-content}

Använd läget **[!UICONTROL Code your own]** för att importera råformat HTML och koda ditt e-postinnehåll.

>[!CAUTION]
>
>Den här metoden kräver kunskaper om HTML.

1. På hemsidan [E-posta Designer](get-started-email-designer.md) väljer du **[!UICONTROL Code your own]**.

   ![Skärmbild som visar alternativet &quot;Koda din egen&quot; på Designer e-posthemsida](assets/code-your-own.png){zoomable="yes"}

1. Ange eller klistra in HTML-råkod på huvudarbetsytan.

1. Använd den vänstra rutan för att komma åt personaliseringsfunktioner. [Läs mer](../personalization/gs-personalization.md)

   ![Skärmbild som visar kodredigeraren med anpassningsalternativ i den vänstra rutan](assets/code-editor-personalization.png){zoomable="yes"}

1. Klicka på knappen **[!UICONTROL Simulate content]** om du vill förhandsgranska meddelandets design och anpassning med testprofiler. [Läs mer](../preview-test/preview-test.md)

1. När koden är klar klickar du på **[!UICONTROL Save & close]** för att gå tillbaka till fönstret där meddelandet skapades och slutföra meddelandet.
